# AdSwitcher

[Wiki](https://github.com/tkyaji/AdSwitcher/wiki)

複数のAdNetworkを切り替えて、広告を表示するライブラリです。<br>
設定により、確率、またはローテーションにて表示する広告を選択します。<br>
設定はURL、または文字列で指定したJSONファイルから読み込みます。


## 対応プラットフォーム
* iOS
* Android
* Unity

## 対応サイズ
* バナー(320x50)
* バナー(320x100)
* バナー(300x250)
* インタースティシャル (動画を含む)
